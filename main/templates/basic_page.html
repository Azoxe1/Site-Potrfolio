Практика в Re-action

Стажировка проходит с 25.09.2023, в режиме фултайм работы, в команде
из 6-ти разработчиков. Сервис автоматической рассылки резюме
по вакансиям. К сожалению, нам не разрешается делиться фрагментами кода.
Последние задачи:

Форма обратной связи

    Описание функционала:
  • Форма обратной связи отправляющая запрос, на основании
    данных указанных пользователем. При создании заявки, на почту администратора
    падает письмо, с подробным описанием заявки (данные пользователя).
    Форма сохраняет данные в БД,с учётом временного пояса юзера.
    Использованные технологии:
  • GIT, Python, SQL, DRF, HTML, CSS

Open API

    Описание задачи:
  • На основании существующего функционала написание Open API. Написание
    дополнительного функционала для логгирования, авторизации,
    регистрации юзеров.
    Использованные технологии:
  • GIT, Python, SQL, DRF, OpenAPI

Взаимодействие с HH

    Различные задачи, по взаимодействию с API HH.ru.
    Примеры задач:
  • Создание сохраненного поиска у авторизованного через токен HH юзера
  • Вывод последних запросов юзера на НН
  • Сбор и сохранение информации по найденным вакансиям для юзера
    Использованные технологии:
  • GIT, Python, SQL, DRF, OpenAPI
